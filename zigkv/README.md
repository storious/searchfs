# zigkv

zigkv is a learning-oriented in-memory KV cache written in Zig.

It is part of systemlab and focuses on low-level systems topics:

- manual memory management
- hash table based KV storage
- TTL expiration
- cache eviction
- TCP protocol design
- persistence with WAL
- mmap and log-structured storage

zigkv is not a production Redis replacement.
It is a systems programming lab.
